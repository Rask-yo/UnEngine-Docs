# Game Rules

## Blacklisted Mods

* Any hacked clients
* Any ghost clients
* Any injectable clients
* Minimap with player entity mod&#x20;
* X-ray mod / texture pack
* Minebot mod
* Inventory Tweaks mod
* Chest ESP mod
* Macros / Keybind mods
* Auto-Clicker mod
